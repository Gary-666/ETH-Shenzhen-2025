# 项目概述
## 🚀 ChainPulse - 链上脉搏

> **AI驱动的区块链数据分析助手：NFT情绪预测 + 链上地址分析**

## 🎯 项目愿景

ChainPulse 致力于链上数据分析，通过AI技术为用户提供实时的区块链洞察、情绪分析和风险评估，让每个人都能轻松理解复杂的链上数据。
